## Project setup
- You will need to install [node](https://nodejs.org/en/) and yarn.
```bash
npm install -g yarn
```

- Clone the repo.

- Install dependencies:
```bash
yarn install
```

- Start app:
```
yarn start
```

## Test

- Run the test:
```
yarn test
```
